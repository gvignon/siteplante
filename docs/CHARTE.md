# 🎨 Charte du site

Référence de style pour garder un site cohérent. Claude Code s'y réfère quand il code
l'apparence.

## Ton éditorial
Clair, prudent, pédagogique. On s'adresse à des débutants en cueillette. La sécurité
prime toujours sur l'enthousiasme.

## Couleurs (suggestion nature)
- Vert principal : `#3A7D44` (titres, en-tête)
- Vert clair fond : `#F2F7F2`
- Terre / accent : `#A9744F`
- Texte : `#2B2B2B`
- Alerte / précaution : `#C0392B`

## Typographie
- Titres : police sans-serif lisible (ex. system-ui, "Segoe UI").
- Corps : même famille, taille confortable (16–18px), interligne aéré.

## Composants récurrents
- **Bandeau d'avertissement** présent sur chaque fiche : fond clair, bordure rouge,
  texte « En cas de doute sur l'identification, ne pas consommer. »
- **Carte plante** (dans la liste) : photo, nom commun, nom latin en italique, badge
  saison.
- **Pied de page** : rappel que le site est informatif + lien sources.

## Accessibilité
- Texte alternatif sur chaque image (nom de la plante).
- Contraste suffisant entre texte et fond.
- Site lisible sur mobile.
